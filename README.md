# Source base of vue3 + typescript, bundle vitejs

`instance-code/vuets` is a source base 

## Install

### Dev
``` bash
$ git clone git@github.com:instance-code/vuets.git
$ yarn install 
$ yarn dev 
```

### Production
``` bash
$ git clone git@github.com:instance-code/vuets.git
$ yarn install 
$ yarn build
$ yarn start 
```
